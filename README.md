# Smart Team Builder

대학 수업 조편성 자동화 도구. 단일 HTML 파일로 동작.

## 사용
브라우저에서 다음 URL로 접속:
**https://kyjung1310-lab.github.io/team-maker/**

## 기능
- 학생 명단 입력 (텍스트·CSV·엑셀)
- 사전 지정 팀/짝 자동 배치
- 랜덤 자동 매칭
- 드래그앤드롭으로 팀/짝 변경 (학생 위 드롭 시 swap)
- 학생 직접 추가·편집·삭제
- JSON 저장/불러오기 (다른 PC 이전)
- LocalStorage 자동 백업
- CSV 다운로드 / 학기 출석부 엑셀 생성
